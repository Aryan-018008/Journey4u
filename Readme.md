# 🌍 Journey4u — Travel Agency Website

Journey4u is a modern, responsive travel agency website designed to showcase destinations, travel packages, and provide a seamless user experience for customers planning their trips.

---

# Deployment
 Journey4u is live at -  https://journey4u.netlify.app/

## 🚀 Features

* 🏠 **Home Page**

  * Hero section with call-to-action
  * Featured destinations
  * Testimonials slider
  * Search functionality

* 📖 **About Us**

  * Company story
  * Core values
  * Brand identity

* 🌍 **Destinations**

  * Filterable destinations (Asia, Europe, Beach, Adventure)
  * Dynamic UI cards

* 💼 **Packages**

  * Travel packages with pricing
  * Highlights and inclusions

* 📞 **Contact Page**

  * Interactive enquiry form
  * Client-side validation
  * Contact information

* 📱 **Responsive Design**

  * Mobile-friendly navigation
  * Hamburger menu

* 🔗 **Footer**

  * Quick links
  * Contact details
  * Social media integration (Instagram, X and  Facebook)

## ⚙️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/Aryan-018008/journey4u.git
```

2. Navigate to the project folder:

```
cd journey4u
```

3. Open `index.html` in your browser
